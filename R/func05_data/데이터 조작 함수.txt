데이터 추출
filter(데이터, 조건)

데이터 정렬
arrange(데이터, 컬럼명, ...) - 컬럼명순으로 오름차순
arrange(데이터, desc(컬럼명), ...) - 내림차순

데이터 검색
select(데이터, 컬럼명, ...) - 컬럼 지정
select(데이터, 컬럼명1:컬럼명2) - 컬럼 범위 지정
select(데이터, -(컬럼명1:컬럼명)) - 컬럼 범위 제외

데이터에 열 추가
mutate(데이터, 열이름=입력, ...)

그룹화
group_by(데이터, 그룹기준)

요약통계치 계산
summarise(),summarize()


library("ggplot2")
ggplot()

파이프 연산자 %>% : 데이터를 뒤 함수에 입력 : 연속 파이프 가능
	- 주의 : 파이프 연산자로 연속으로 이을때 파이프 연산자 앞에서 엔터를 치면 명령이 끝난 것으로 컴퓨터가 오류. 파이프 연산자 뒤에서 엔터를 치자


데이터 프레임 join
inner_join(데이터, 데이터, by = '열이름')
left_join(데이터, 데이터, by = '열이름')
right_join(데이터, 데이터, by = '열이름')
full_join(데이터, 데이터, by = '열이름')

bind_rows(데이터, 데이터)
bind_cols(데이터, 데이터)

재구조화 함수
형식 변환 (긴 형식 -> 넓은 형식)
dcast(데이터, ~, 함수)

형식 변환 (넓은 형식 -> 긴 형식)
melt(데이터, ~, 함수)

acast()

transform
(
	데이터,
	바꾸려는열=new값
)

sapply() - 자료구조로 데이터 반환
lapply() - 리스트로 데이터 반환
